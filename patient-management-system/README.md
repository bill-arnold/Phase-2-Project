## Introduction

Patient Management System is a web application built with React that allows healthcare professionals to manage patient records, schedule appointments, and receive reminders. It utilizes a simple API to store patient data.

## Features

- View a list of patients with their details.
- Add new patients with their name, date of birth, and appointment date.
- Delete patients from the list.
- Receive reminders for upcoming appointments.

# Getting Started

# Prerequisites

- Node.js and npm installed on your machine.

 Installation

1. Clone the repository to your local machine:

   ```bash
   git clone git@github.com:bill-arnold/Phase-2-Project.git
   ```

2. Navigate to the project directory:

   ```bash
   cd patient-management-system
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

Usage

1. Start the development server:

   ```bash
   npm start
   ```

2. Open your browser and go to [http://localhost:3000](http://localhost:3000) to access the application.

3. Use the application to add, view, and delete patient records.

Technologies Used

- **React:** JavaScript library for building user interfaces.
- **Axios:** HTTP client for making requests to the API.
- **JSON Server:** Simple REST API server for storing patient data.
